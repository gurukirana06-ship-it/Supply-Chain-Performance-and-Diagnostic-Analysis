# Supply Chain Performance & Diagnostic Analytics Dashboard

## Project Overview

This project focuses on analyzing end-to-end supply chain operations using Power BI, SQL, and Excel. The objective was to evaluate inventory performance, supplier efficiency, logistics operations, product quality, and overall supply chain health through interactive dashboards and diagnostic analytics.

The project helps identify operational bottlenecks, monitor key performance indicators (KPIs), and support data-driven decision-making for supply chain optimization.

## Dataset Overview

The dataset contains supply chain information related to:

- Product Categories and SKUs
- Inventory Levels and Stock Availability
- Supplier Performance Metrics
- Logistics and Shipping Information
- Sales and Revenue Data
- Quality Inspection Results
- Production and Demand Metrics

## What I Did

- Cleaned and transformed raw data using Excel, SQL, and Power Query.
- Performed data normalization and built relationships using Power BI data modeling.
- Created DAX measures and KPIs to track supply chain performance.
- Developed interactive dashboards for Product, Inventory, Supplier, Logistics, and Executive-level analysis.
- Implemented drill-through pages, bookmarks, and dynamic filtering for detailed investigation.
- Analyzed inventory risks, supplier performance, logistics costs, and quality metrics.
- Identified critical supply chain bottlenecks and provided business recommendations.

## Conclusion

The analysis identified inventory shortages, supplier variability, logistics inefficiencies, and quality management challenges as the primary supply chain bottlenecks. The dashboard provides actionable insights that can help improve inventory availability, operational efficiency, and overall supply chain performance.
